Fields

Each document has the following fields:

Property Name Type Description
name String The qualified name
simpleName String The simple name (without qualifiers)
params Array An array of parameter names
kind String The kind of symbol. One of:
  • method
  • function
  • class
  • exception
path String The path of the documentation, relative to https://docs.python.org/[version]/library/
summaryHtml String A summary of the function/method/class/exception, in HTML format
enclosingModule String The name of the module that contains this code object

Example

{
  "name": "set.add",
  "simpleName": "add",
  "params": ["elem"],
  "kind": "method",
  "path": "stdtypes.html#set.add",
  "summaryHtml": "<p>Add element <em>elem</em> to the set.</p>",
  "enclosingModule": "stdtypes"
}
