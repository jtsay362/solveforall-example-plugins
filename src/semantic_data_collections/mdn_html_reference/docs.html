Fields

Each document has the following fields:

Property Name Type Description
name String The name of the element
summary String A description of the element
html5Only Boolean True if the element is new in HTML5
attributes Array An array of supported attributes. Each attribute object has the following properties:
  • name: The name of the attribute
  • summary: A summary of the usage of the attribute
  • html5Only: True if the attribute is was added in HTML5 to an existing element

Example

{
  "name" : "audio",
  "summary" : "The HTML <audio> element is used to embed sound content in documents...",
  "html5Only" : true, 
  "attributes" : [
    { 
      "name" : "autoplay",
      "summary" : "A Boolean attribute...",
      "html5Only" : false
    },
    ...
  ]
}
