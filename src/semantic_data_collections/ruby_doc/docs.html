Fields

Each document has the following fields:

Property Name Type Description
name String The simple name. Can be searched by rawName (not analyzed) or name (analyzed).
fullName String The fully qualified name. Can be searched by rawFullName (not analyzed) or fullName (analyzed).
fullNameTokens Array of Strings An array of tokens in the full name. Can be searched by rawFullNameTokens (not analyzed) or fullNameTokens (analyzed).
project String The name of the project / library
kind String The kind of symbol. One of:
  • "module"
  • "class"
  • "method"
  • "attribute"
  • "constant"
parent String The fully qualified name of the parent context of the code object (either a module or a class). null for top-level classes and modules.
uri String The path of the documentation, which may contain the string ${VERSION} as a placeholder for the version of the library
summaryHtml String A summary of the code object, in HTML format
visibility String "public" if the code object is externally visible

Context kinds (classes and modules) also have the following fields:

Property Name Type Description
extends Array of Strings An array of fully-qualfied names of contexts this context extends
includes Array of Strings An array of fully-qualfied names of contexts this context includes
methods Array of Objects An array of objects representing each method. Each object has the following properties:
  • name: The simple name of the method
  • summaryHtml: A brief summary of the method, in HTML format
  • visibility: "public" if the method is externally visible
  • params: A comma separated list of parameters
  • uri: The URI pointing to the documentation of the method. May also contain the ${VERSION} placeholder.
attributes Array of Objects An array of objects representing each attribute. Each object has the following properties:
  • name: The simple name of the attribute
  • summaryHtml: A brief summary of the attribute, in HTML format
  • visibility: "public" if the attribute is externally visible
  • uri: The URI pointing to the documentation of the attribute. May also contain the ${VERSION} placeholder.
constants Array of Objects An array of objects representing each constant. Each object has the following properties:
  • name: The simple name of the constant
  • summaryHtml: A brief summary of the constant, in HTML format
  • value: The value of the constant
  • uri: The URI pointing to the documentation of the method. May also contain the ${VERSION} placeholder.

Classes also have the following field:

Property Name Type Description
superClass String The fully-qualfied name of the superclass (parent class) of the class

Methods also have the following field:

Property Name Type Description
params String The parameters for the method, separated by commas

Constants also have the following field:

Property Name Type Description
value String The value of the constant

Examples

A class
{
  "name": "Hash",
  "fullName": "Hash",
  "project": "ruby",
  "kind": "class",
  "fullNameTokens": [
    "Hash"
  ],
  "parent": null,
  "includes": [
    "Enumerable"
  ],  
  "extends": [],
  "superClass": "Object",  
  "visibility": "public",  
  "summaryHtml": "<p>A <a href=\"http://www.ruby-doc.org/core-${VERSION}/Hash.html\">Hash</a> is a dictionary-like collection of unique\nkeys and their values...",  
  "uri": "http://www.ruby-doc.org/core-${VERSION}/Hash.html",
  
  "methods": [
    ...
    {
      "name": "fetch",
      "summaryHtml": "<p>Returns a value from the hash for the given key. If the key can&#39;t be\nfound, there are several options: ...",
      "params": "(key) {| key | block }   -> obj",
      "visibility": "public",
      "uri": "http://www.ruby-doc.org/core-${VERSION}/Hash.html#method-i-fetch"
    },
    ...
  ],    
  "attributes": [],      
  "constants": []
}
A method
{
  "name": "keys",
  "fullName": "Hash#keys",
  "project": "ruby",
  "kind": "method",
  "fullNameTokens": [
    "Hash",
    "keys"
  ],
  "parent": "Hash",
  "visibility": "public",  
  "summaryHtml": "<p>Returns a new array populated with the keys from this hash...",  
  "params": "    -> array",
  "uri": "http://www.ruby-doc.org/core-${VERSION}/Hash.html#method-i-keys"
}
