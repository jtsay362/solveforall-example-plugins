Fields

Each document has the following fields:

Property Name Type Description
name String The name of the property, function, at-rule, pseudo-class, pseudo-element, data type, or general topic
summary String A brief description of the item
syntax String Formal syntax, if the item is a property
metaProperties Object Name/value pairs about the property, if applicable
values Array of Objects If the item is a property, an array of objects that have the following properties:
  • name: The name of the value
  • summary: A summary of the usage of the value
mdnUri String URL of documentation on Mozilla Developer Network (not always available)
w3cUri String URL of documentation on W3C (not always available)
webPlatformUri String URL of documentation on WebPlatform.org (not always available)

Example

{
  "name": "margin-left",
  "summary": "The margin-left CSS property of an element sets the margin space required on the left side...",
  "syntax": "<length> | <percentage> ...",
  "metaProperties": {
    "Initial value": "0",
    ...
  }, 
  "values": [
    {
      "name": "length",
      "summary": "Specifies a fixed length ..."
    },
    ...
  ],
  "mdnUri": "https://developer.mozilla.org/en-US/docs/Web/CSS/margin-left",
  "w3cUri": "http://www.w3.org/wiki/CSS/Properties/margin-left",
  "webPlatformUri": "http://docs.webplatform.org/wiki/css/properties/margin-left"
}
