Fields

Each document has the following fields:

Property Name Type Description
name String The word
senses Array An array of senses. Each sense object has the following properties:
  • partOfSpeech: The part of speech of the sense
  • definition: The definition of the sense
  • synonyms: An array of words that are synonyms of the word, used in the particular sense

Example

{
  "name": "awesome",
  "senses": [
    {
      "partOfSpeech": "adjective satellite",
      "definition": "inspiring awe or admiration or wonder",
      "synonyms": [ "amazing", "awe-inspiring", ... ]
    }
  ]
}
