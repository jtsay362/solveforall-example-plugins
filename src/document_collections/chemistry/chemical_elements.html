Fields

Each document has the following fields:

Property Name Type Description
name String The full name of the element
symbol String The chemical symbol of the element
number Number The atomic number of the element
group Number The group number of the element in the periodic table
period Number The period number of the element in the periodic table
series Number The series number of the element in the periodic table
block String The block of the element in the periodic table
eleconfig String The ground state electron configuration of the element
description String A brief description of the element
mass Number The relative atomic mass of the element (Ar)
density Number The density of the element in g/cm3 at 295 K
oxidation states String The comma-separated oxidation states of the element
tmelt Number The melting temperature of the element, in K, at 1 atm
tboil Number The boiling temperature of the element, in K, at 1 atm
eleneg Number The electronegativity of the element, on the Pauling scale
eleaffin Number The electron affinity of the element, in eV
ionenergy Array of Numbers The ionization energies of the element, in eV
covrad Number The covalent radius of the element (rcov), in angstroms
vdwrad Number The Van der Waals radius of the element (rw), in angstroms
isotopes Array of Objects An array of objects that have the following properties:
  • massnumber: The mass number of the isotope
  • mass: The atomic mass of the isotope, in atomic mass units
  • abundance: The abundance of the isotope, relative to 1.0
