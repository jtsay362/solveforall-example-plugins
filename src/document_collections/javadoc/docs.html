Fields

Each document has the following fields:

Property Name Type Description
class String The simple class name
qualifiedClass String The qualified class name
name String The simple name
annotations Array An array of annotations
modifiers Array An array of modifiers ("public", "static", etc.)
description String A description of the class/enum/annotation/method
since String Which JDK this first appeared in (may not be present)
path String The path of the documentation, relative to http://docs.oracle.com/javase/[version]/docs/api/
kind String The kind of symbol. One of:
  • class
  • enum
  • interface
  • annotation
  • method

Class Fields

For classes, enums, and interfaces, the document also contains the following properties:

Property Name Type Description
superClass String The fully-qualified name of the superclass
implements Arrays An array of fully-qualified interface names this class or enum implements, or this interface extends
methods Array An array containing these properties of methods (see below):
  • name
  • modifiers
  • params (without the description property)
  • returnType
  • path

Methods Fields

For methods, the following properties are also available:

Property Name Type Description
params Array An array of parameter descriptors with the following properties:
  • name
  • type
  • description (may not be present)
returnType String The return type of the method
throws Array An array of Exception subclass names declared by the method (subclasses of java.lang.RuntimeException are typically not declared)
returns String The description of the return value of the method (may not be present)

Recognition Keys

Classes, enums, interfaces, and annotations will be mapped to the com.solveforall.programming.java.JdkClass recognition key, while methods will be mapped to the com.solveforall.programming.java.JdkMethod recognition key.
