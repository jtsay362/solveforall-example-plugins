Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The entire tracking number
recognitionLevel Number The recognition level. 1.0 if a valid tracking number, 0.25 if invalid.
valid Boolean True if the tracking number is valid (has a valid check digit and service type)
validCheckDigit String The corrected check digit
accountNumber String The account number
invoiceNumber String The invoice number
packageNumber String The package number
serviceType String The service type. One of
  • Next Day Air
  • Second Day Air
  • Ground
  • (Invalid)

Example

{
  "com.solveforall.recognition.business.UPSTrackingNumber": [
    {
      "valid": true,
      "accountNumber": "AE9959",
      "matchedText": "1ZAE99590353802974",
      "validCheckDigit": "4",
      "recognitionLevel": 1,
      "invoiceNumber": "53802",
      "serviceType": "Ground",
      "packageNumber": "97"
    }
  ]
}
