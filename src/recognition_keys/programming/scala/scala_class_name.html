Fields

Each recognition result has the following fields:

Property Name Type Description
recognitionLevel Number The recognition level. Always 1.0.
packageName String The package name of the Scala class
simpleClassName String The simple name of the Scala class (without the package)
fullyQualifiedClassName String The fully qualified name of the Scala class (including the package)

Example

{
  "com.solveforall.recognition.programming.scala.ScalaClassName": [
    {
      "recognitionLevel": 1.0,
      "packageName": "scala.concurrent",
      "simpleClassName": "Future",
      "fullyQualifiedClassName": "scala.concurrent.Future"
    }
  ]
}
