Fields

Each recognition result has the following fields:

Property Name Type Description
recognitionLevel Number The recognition level. 1.0 for classes in known Java packages, 0.5 otherwise.
packageName String The package name of the Java class
simpleClassName String The simple name of the Java class (without the package)
fullyQualifiedClassName String The fully qualified name of the Java class (including the package)

Example

{
  "com.solveforall.recognition.programming.java.JavaClassName": [
    {
      "recognitionLevel": 1.0,
      "packageName": "java.util",
      "simpleClassName": "Map",
      "fullyQualifiedClassName": "java.util.Map"
    }
  ]
}
