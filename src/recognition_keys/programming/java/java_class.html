Fields

Each document has the following fields:

Property Name Type Description
class String The simple class name
qualifiedClass String The qualified class name
name String The simple name
annotations Array An array of annotations
modifiers Array An array of modifiers ("public", "static", etc.)
description String A description of the class/interface/enum/annotation
since String Which JDK this first appeared in (may not be present)
path String The path of the documentation, relative to http://docs.oracle.com/javase/[version]/docs/api/
kind String The kind of symbol. One of:
  • "class"
  • "enum"
  • "interface"
  • "annotation"
superClass String The fully-qualified name of the superclass
implements Arrays An array of interface names this class or enum implements, or this interface extends
constructors Array An array containing these properties of constructors:
  • modifiers
  • params (without the description property)
  • description
  • path
methods Array An array containing these properties of methods:
  • name
  • modifiers
  • params (without the description property)
  • returnType
  • description
  • returns
  • path
