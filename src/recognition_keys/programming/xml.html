Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched XML string
recognitionLevel Number The recognition level:
  • 0.2 if the query contains <, >, or / character(s), but pretty far from valid XML
  • 0.6 if the query contains text that is close to valid XML
  • 1.0 if the query contains valid XML
isValid boolean true if valid XML, false otherwise
parseErrorMessage String If the XML-like text was not valid, a String indicating the reason for the parsing error, otherwise null

Example

{
  "com.solveforall.recognition.programming.Xml": [
    {
      "matchedText": "<hi a=\"b\">yo</hi>",
      "recognitionLevel": 1.0,
      "parseErrorMessage": null,
      "isValid": true,
    }
  ]
}

Notes

This recognizer does not currently support XML declarations (such as <?xml version="1.0 encoding="UTF-8"?>).
