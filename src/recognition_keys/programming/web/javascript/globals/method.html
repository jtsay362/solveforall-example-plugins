Fields

Each recognition result has the following fields:

Property Name Type Description
qualifiedName String The name of the method, qualified by the class name
simpleName String The qualified name of the method
class String The name of the class the method belongs to
syntax String Syntax on how to call the method
parametersHtml String A description of the parameters, in HTML format
returnValueDescriptionHtml String A description of the return value, in HTML format
errorsThrownHtml String A description of the errors thrown, in HTML format
summaryHtml String A summary of the method, in HTML format
descriptionHtml String A description of the method, in HTML format
deprecated Boolean True if the method is deprecated
ecma6 Boolean True if the method is only available in ECMAScript 6
nonStandard Boolean True if the method is non-standard
kind String Always "Method"
url String A URL pointing to the original documentation on Mozilla Developer Network
