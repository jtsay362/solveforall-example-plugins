Fields

Each recognition result has the following fields:

Property Name Type Description
qualifiedName String The name of the property, qualified by the class name
simpleName String The simple name of the property
class String The name of the class the property belongs to
summaryHtml String A summary of the property, in HTML format
descriptionHtml String A description of the property, in HTML format
deprecated Boolean True if the property is deprecated
ecma6 Boolean True if the property is only available in ECMAScript 6
nonStandard Boolean True if the property is non-standard
kind String Always "ClassProperty"
url String A URL pointing to the original documentation on Mozilla Developer Network

Example

{
  "com.solveforall.recognition.programming.web.javascript.globals.ClassProperty": [
    {
      "simpleName": "PI",
      "qualifiedName": "Math.PI",
      "class": "Math",      
      "summaryHtml": "The <code><strong>Math.PI</strong></code> property represents the ratio of the circumference of a circle to its diameter, approximately 3.14159:",
      "descriptionHtml": "Because <code>PI</code> is a static property of <code>Math</code>, you always use it as <code>Math.PI</code>, rather than as a property of a <code>Math</code> object you created (<code>Math</code> is not a constructor).",
      "deprecated": false,
      "ecma6": false,
      "nonStandard": false,      
      "kind": "ClassProperty",
      "url": "https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/PI"
    }
  ]
}
