Fields

Each recognition result has the following fields:

Property Name Type Description
qualifiedName String The name of the function
simpleName String The name of the function
class String Always null
syntax String Syntax on how to call the function
parametersHtml String A description of the parameters, in HTML format
returnValueDescriptionHtml String A description of the return value, in HTML format
errorsThrownHtml String A description of the errors thrown, in HTML format
summaryHtml String A summary of the function, in HTML format
descriptionHtml String A description of the function, in HTML format
deprecated Boolean True if the function is deprecated
ecma6 Boolean True if the function is only available in ECMAScript 6
nonStandard Boolean True if the function is non-standard
kind String Always GlobalFunction
url String A URL pointing to the original documentation on Mozilla Developer Network
