Fields

Each recognition result has the following fields:

Property Name Type Description
qualifiedName String The name of the class
simpleName String The name of the class
class String The name of the class
constructor Object If the class has a constructor, an object with the property params, that is an array of objects with the following properties:
  • name: The name of the constructor parameter
  • descriptionHtml: The description of the constructor parameter, in HTML format
classProps Array An array of objects, describing the class properties, with the following keys:
  • simpleName
  • summaryHtml
  • deprecated
  • ecma6
  • nonStandard
  • url
See the documentation for com.solveforall.recognition.programming.web.javascript.globals.ClassProperty for more details.
classMethods Array An array of objects, describing the class methods, with the following keys:
  • simpleName
  • summaryHtml
  • syntax
  • parametersHtml
  • deprecated
  • ecma6
  • nonStandard
  • url
See the documentation for com.solveforall.recognition.programming.web.javascript.globals.ClassMethod for more details.
props Array An array of objects, describing the instance properties, with the following keys:
  • simpleName
  • summaryHtml
  • deprecated
  • ecma6
  • nonStandard
  • url
See the documentation for com.solveforall.recognition.programming.web.javascript.globals.Property for more details.
methods Array An array of objects, describing the instance methods, with the following keys:
  • simpleName
  • summaryHtml
  • syntax
  • parametersHtml
  • deprecated
  • ecma6
  • nonStandard
  • url
See the documentation for com.solveforall.recognition.programming.web.javascript.globals.Method for more details.
summaryHtml String A summary of the class, in HTML format
descriptionHtml String A description of the class, in HTML format
deprecated Boolean True if the class is deprecated
ecma6 Boolean True if the class is only available in ECMAScript 6
nonStandard Boolean True if the class is non-standard
kind String Always "Class"
url String A URL pointing to the original documentation on Mozilla Developer Network
