Fields

Each recognition result has the following fields:

Property Name Type Description
qualifiedName String The name of the property
simpleName String The name of the property
class String Always null
summaryHtml String A summary of the property, in HTML format
descriptionHtml String A description of the property, in HTML format
deprecated Boolean True if the property is deprecated
ecma6 Boolean True if the property is only available in ECMAScript 6
nonStandard Boolean True if the property is non-standard
kind String Always "GlobalProperty"
url String A URL pointing to the original documentation on Mozilla Developer Network

Example

{
  "com.solveforall.recognition.programming.web.javascript.globals.GlobalProperty": [
    {
      "qualifiedName": "Infinity",
      "simpleName": "Infinity",
      "class": null,                
      "summaryHtml": "The global <code><strong>Infinity</strong></code> property is a numeric value representing infinity.",
      "descriptionHtml": "<code>Infinity</code> is a property of the <em>global object</em>, i.e. it is a variable in global scope.",      
      "kind": "GlobalProperty",
      "deprecated": false,
      "ecma6": false,
      "nonStandard": false,
      "url": "https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Infinity"
    }
  ]
}
