Fields

Each document has the following fields:

Property Name Type Description
name String The simple name. Can be searched by rawName (not analyzed) or name (analyzed).
fullName String The fully qualified name. Can be searched by rawFullName (not analyzed) or fullName (analyzed).
fullNameTokens Array of Strings An array of tokens in the full name. Can be searched by rawFullNameTokens (not analyzed) or fullNameTokens (analyzed).
project String The name of the project / library
kind String Always "method"
params String The parameters for the method, separated by commas
uri String The path of the documentation, which may contain the string ${VERSION} as a placeholder for the version of the library
summaryHtml String A summary of the method, in HTML format
visibility String "public" if the method is externally visible

Example

{
  "com.solveforall.recognition.programming.ruby.Method": [
    {
      "name": "keys",
      "fullName": "Hash#keys",
      "project": "ruby",
      "kind": "method",
      "fullNameTokens": [
        "Hash",
        "keys"
      ],
      "parent": "Hash",
      "visibility": "public",  
      "summaryHtml": "<p>Returns a new array populated with the keys from this hash...",  
      "params": "    -> array",
      "uri": "http://www.ruby-doc.org/core-${VERSION}/Hash.html#method-i-keys"
    },
    ...
  ]
}
