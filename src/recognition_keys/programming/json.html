Fields

Each recognition result has the following fields:

Property Name Type Description
formattedString String The formatted JSON string, with an indentation level of 2 spaces.
matchedText String The matched JSON string
recognitionLevel Number The recognition level. Always 1.0.

Example

{
  "com.solveforall.recognition.programming.Json": [
    {
      "matchedText": "{\"hi\": [1, 2, 3], \"ho\": false }",
      "formattedString": "{\n \"hi\": [\n 1,\n 2,\n 3\n ],\n \"ho\": false\n}"}",
      "recognitionLevel": 1.0
    }
  ]
}
