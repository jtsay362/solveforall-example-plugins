Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched date or date range.
recognitionLevel Number The recognition level. Always 0.5.
start.year Number The year, or if a date range, the starting year.
start.month Number The month, or if a date range, the starting month.
start.dayOfMonth Number The day of the month, or if a date range, the starting day of the month.
start.hour Number The hour, or if a date range, the starting hour. Starts at 0 for midnight, up to 23 for 11:00 pm.
start.minute Number The minute, or if a date range, the starting minute.
start.second Number The second, or if a date range, the starting second.
end.year Number If a date range, the ending year. Not present if only a single date.
end.month Number If a date range, the ending month. Not present if only a single date.
end.dayOfMonth Number The day of If a date range, the ending day of the month. Not present if only a single date.
end.hour Number If a date range, the ending hour. Starts at 0 for midnight, up to 23 for 11:00 pm. Not present if only a single date.
end.minute Number If a date range, the ending minute. Not present if only a single date.
end.second Number If a date range, the ending second. Not present if only a single date.

Example

{
  "com.solveforall.recognition.date.DateRange": [
    {
      "matchedText": "Jan 5, 2013 11:00 pm to Jan 7, 2013 2:30 am",
      "recognitionLevel": 0,
      "start": {
        "year": 2013,
        "month": 1,
        "dayOfMonth": 5,
        "hour": 23,
        "minute": 0,
        "second": null
      },
      "end": {
        "year": 2013,
        "month": 1,
        "dayOfMonth": 7,
        "hour": 2,
        "minute": 30,
        "second": null
      }
    }
  ]
}
