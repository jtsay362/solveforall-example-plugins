Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched phone number. May include a +1 or 1 prefix. May also contain parentheses, dashes, periods, or spaces.
recognitionLevel Number The recognition level. Always 1.0.
areaCode String The three digit area code of the phone number.
numberWithoutAreaCode String The phone number without the area code. May also contain parentheses, dashes, periods, or spaces.

Example

{
  "com.solveforall.recognition.phone.UsPhoneNumber": [
    {
      "matchedText": "1(856)555-2851",
      "recognitionLevel": 1.0,
      "areaCode": "856",
      "numberWithoutAreaCode": "555-2851"
    }
  ]
}
