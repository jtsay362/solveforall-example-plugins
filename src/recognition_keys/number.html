Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched number, as a String.
recognitionLevel Number The recognition level. Always 0.5.
doubleValue Number The matched number, as a Number.

Example

{
  "com.solveforall.recognition.Number": [
    {
      "matchedText": "-6,477,032.667",
      "recognitionLevel": 0.5,
      "doubleValue": -6477032.667,
    }
  ]
}

Notes

This recognizer accepts leading minus signs and ignores commas. It treats a period as a decimal point.
