Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched PGN text
recognitionLevel Number The recognition level:
  • 0.2 if there is only a single ply and no tags
  • 0.7 if there is only a single ply and at least one tag
  • 1.0 if there are at least two plies
numPlies Number The total number of plies (moves) in the game
date String The date of the game
site String The name of the site of the game
event String The name of the event
round String The name of the round
comment String Additional comments
blackPlayerName String The name of the Black player
blackElo Number The Elo rating of the Black player. 0 if unknown.
whitePlayerName String The name of the White player
whiteElo Number The Elo rating of the White player. 0 if unknown.
gameResult String The result of the game. One of:
  • 1 - 0
  • 0 - 1
  • 1/2 - 1/2
