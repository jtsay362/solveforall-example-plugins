Fields

Each recognition result has the following fields:

Property Name Type Description
hashTag String The hash tag, including the leading # sign
tag String The hash tag, not including the leading # sign
recognitionLevel Number The recognition level. Always 1.0.

Example

{
  "com.solveforall.recognition.socialmedia.twitter.HashTag": [
    {
      "hashTag": "#fail",
      "tag": "fail",
      "recognitionLevel": 1.0
    }
  ]
}
