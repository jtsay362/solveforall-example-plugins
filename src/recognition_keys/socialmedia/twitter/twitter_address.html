Fields

Each recognition result has the following fields:

Property Name Type Description
address String The address, not including the leading @ sign
addressWithAt String The address, including the leading @ sign
recognitionLevel Number The recognition level. Always 1.0.

Example

{
  "com.solveforall.recognition.socialmedia.twitter.Address": [
    {
      "addressWithAt": "@solveforall",
      "address": "solveforall",
      "recognitionLevel": 1.0
    }
  ]
}
