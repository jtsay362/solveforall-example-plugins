Source

Fields

Each recognition result has the following fields:

Property Name Type Description
code String The CPT / HCPCS code
description String A description of the procedure
costs Object A mapping from localities to the US dollar amount reimbursed by Medicare. Localities are generally the capitalized two-letter abbreviations for states, with the following exceptions:
  • CA (North): Northern California
  • CA (South): Southern California
  • MO (General American): Missouri under General American insurance
  • NY (West): West New York
  • NY (Empire): New York (Empire)
  • NY (GHI): New York (GHI)
  • max: 60% National Limitation Amount (NLA)
  • median: Median of the 2013 60% Updated Base Fee Amounts from all Medicare Part B Carriers
  • min: 60% Payment Floor
