Supported elements

  • Decimal constants
  • Parentheses
  • Supported Operators: +, -, *, /, %, &, |, &&, ||, ^, !

Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched arithmetic expression
recognitionLevel Number The recognition level. Always 0.7.

Example

{
  "com.solveforall.recognition.math.ArithmeticExpression": [
    {
      "matchedText": "-6.3 + 2 * (5 % 2)",
      "recognitionLevel": 0.7
    }
  ]
}

Notes

This recognizer only looks for numbers, operators, and parentheses. It does not validate the syntax of expressions.
