Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched part of the query
recognitionLevel Number The recognition level. Always 1.0.
airlineIata String The IATA airline designator of the airline
airlineIcao String The ICAO airline designator of the airline
airlineCallSign String The call sign of the airline
airlineName String The name of the airline, in English
flightNumber String The flight number

Example

{
  "com.solveforall.recognition.travel.Flight": [
    {
      "matchedText": "BR11",
      "recognitionLevel": 1.0,
      "airlineIata": "BR",
      "airlineIcao": "EVA",
      "airlineCallSign": "EVA Air",
      "airlineName": "EVA",
      "flightNumber": "11"
    }
  ]
}
