Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched email address.
recognitionLevel Number The recognition level. Always 0.7.
username String The username (the part before the @).
domain String The domain of the address (the part after the @. May also include a hostname in front.

Example

{
  "com.solveforall.recognition.EmailAddress": [
    {
      "matchedText": "admin@solveforall.com",
      "recognitionLevel": 0.7,
      "username": "admin",
      "domain": "solveforall.com"
    }
  ]
}
