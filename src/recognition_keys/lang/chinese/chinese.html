Fields

Each recognition result has the following fields:

Property Name Type Description
traditional String The word or phrase written using traditional Chinese characters
simplified String The word or phrase written using simplified Chinese characters
pinyin String The pinyin pronunciation of the word or phase
meanings Array of Strings The meanings of the word or phrase
recognitionLevel Number 1.0 if exact match, 0.0 ~ 1.0 for pinyin matches that are off by tones

Example

{
  "com.solveforall.recognition.lang.chinese.Chinese": [
    {
      "meanings": [
        "to decide (to do something)",
        "to resolve",
        "decision",
        "CL:個|个[ge4],項|项[xiang4]",
        "certainly"
      ],
      "matchedText":"jue ding",
      "traditional":"決定",
      "recognitionLevel":0.875,
      "simplified":"决定",
      "pinyin":"jue2 ding4"
    }
  ]
}

Note

The matched text property of the recognition result is not accurate. It will be corrected in the future.
