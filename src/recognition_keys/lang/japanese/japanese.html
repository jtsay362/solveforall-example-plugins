Fields

Each recognition result has the following fields:
Property Name Type Description
matchedText String The matched Japanese text (either hiragana, katakana, kanji, or romaji) in the query
recognitionLevel Number 1.0 if known hiragana, katakana, or kanji. 0.7 if known romaji
kanaOrKanji String The word or phrase written using hiragana, katakana, or kanji

Example

{
  "com.solveforall.recognition.lang.jpn.Japanaese": [
    {
      "matchedText":"taberu",
      "recognitionLevel":0.7,
      "kanaOrKanji":"たべる"
    }
  ]
}

Note

The matched text property of the recognition result is not accurate. It will be corrected in the future.
