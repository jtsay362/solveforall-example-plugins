Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The matched URL.
recognitionLevel Number The recognition level. Always 1.0.
protocol String The protocol, either http or https.
host String The hostname.
path String The path, including a leading /.

Example

{
  "com.solveforall.recognition.WebUrl": [
    {
      "matchedText": "https://solveforall.com/contact",
      "recognitionLevel": 1.0,
      "protocol": "https",
      "host": "solveforall.com",
      "path": "/contact"
    }
  ]
}
