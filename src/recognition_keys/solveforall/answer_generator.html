Fields

Each recognition result has the following fields:

Property Name Type Description
recognitionLevel Number The recognition level, between 0.1 and 0.2, depending on how strongly the answer query matches
id Number (long) The internal ID of the Answer Generator. Used to form URLs.
name String The localized name
iconUrl String The URL of a favicon. May be null.
providerId Number (long) The internal ID of the Content Provider of the Answer Generator. Used to form URLs. May be null if there is no Content Provider set.
providerName String The localized name of the Content Provider of the Answer Generator. May be null if there is no Content Provider set.
categories Array of String An array of category names. May be empty.
publisherUserId Number (long) The internal ID of the publisher of the Answer Generator. Used to form URLs.
created Number (long) The timestamp (number of milliseconds since the epoch) of when the Answer Generator was first created
updated Number (long) The timestamp (number of milliseconds since the epoch) of when the Answer Generator was last updated
numUsers Number (long) The number of users who have added the Answer Generator to their Answer Preferences
meanRating Number (double) The mean rating, on a 0.0 to 5.0 scale, given by users to the Answer Generator
suggestedActivationCode String The suggested (default) activation code, without the ? prefix
