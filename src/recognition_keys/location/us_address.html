Fields

Each recognition result has the following fields:

Property Name Type Description
matchedText String The substring of the query matching an address
recognitionLevel Number The sum of the following values:
  • 0.1 if a street address is present
  • 0.1 if a secondary unit is present
  • 0.3 if a state is present
  • 0.6 if a ZIP code is present
streetAddress String The street address, including the number
secondaryUnit String Secondary unit number, like an apartment or suite number
city String The city
stateName String The full name of the state (e.g. "California")
stateAbbreviation String The abbreviation of the state (e.g. "CA")
zipCode String The ZIP code
countryName String Always "United States"
normalizedFullAddressWithoutCountry String The full address, with all fields normalized, but with the country. The state is abbreviated.

Example

{
  "com.solveforall.recognition.location.UsAddress": [
    {
      "matchedText" : "1600 Pennsylvania Ave NW, Washington, District of Columbia 20500",
      "recognitionLevel" : 1.0,
      "streetAddress" : "1600 Pennsylvania Ave NW",
      "city" : "Washington",
      "stateName" : "District of Columbia",
      "stateAbbreviation" : "DC",
      "zipCode" : "20500",
      "countryName" : "United States",
      "normalizedFullAddressWithoutCountry" : "1600 Pennsylvania Ave NW, Washington, DC 20500"
    }
  ]
}
